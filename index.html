01 SERVICIOS

LO QUE HACEMOS

[01/03] ─ SOFTWARE

Software a Medida

  • Arquitectura enterprise (Clean Architecture, CQRS)
  • APIs y backends de producción
  • Apps móviles offline-first
  • Integración con sistemas empresariales (SAP, ERP)

Diseño y desarrollo de software a medida para operaciones que no pueden fallar, desde backends enterprise hasta apps de campo.

[02/03] ─ HARDWARE

Prototipado & IoT

  • Microcontroladores (ESP32, Arduino)
  • Sistemas embebidos (Raspberry Pi)
  • Sensores, GPS, comunicación inalámbrica
  • Integración física-digital

De la idea al dispositivo funcional: firmware, sensores y dashboards en tiempo real para hardware conectado.

[03/03] ─ IA

IA & Automatización

  • Agentes de IA (Claude, LLMs)
  • Automatización de procesos (RPA)
  • Integración de APIs de IA

Aplicamos IA donde resuelve problemas reales: clasificación automática, agentes operativos y procesos repetitivos.

02 PORTAFOLIO

LO QUE CONSTRUIMOS

Productos propios y en desarrollo. Cada uno combina al menos dos de nuestras tres áreas: software, hardware o IA.

PLANTCARE CONNECT En desarrollo

API de Monitoreo de Plantas

API propia para riego, luz y temperatura en tiempo real, con kits físicos ESP32 + sensores opcionales. Nuestro proyecto insignia: software, hardware e IA en un solo producto.

GITLAB MONITOR Producto propio

Analítica de GitLab

Plataforma de monitoreo en tiempo real para instancias GitLab, con resúmenes ejecutivos generados por IA vía Claude API.

Star Tracker

Raspberry Pi + motores paso a paso para seguimiento astronómico y astrofotografía.

Recepción Biométrica

Control de acceso con lectura biométrica y registro de asistencia sobre Raspberry Pi.

Pañol

Gestión de inventario, préstamos y devoluciones de herramientas y materiales.

WiFi/BLE Sniffer

ESP32-WROVER + OLED con dashboard web embebido para análisis de redes.
